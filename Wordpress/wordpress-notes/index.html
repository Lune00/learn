Built-in Functions

Sur le nom des fonctions

Toutes les fonctions qui commencent par the_ font un appel à echo (ex : the_title()), elles ont une fonction correspondante qui renvoie la valeur commençant par get_ (ex: get_the_title())

Fonctions utiles

  • bloginfo() : renvoie le titre du site
  • the_post()
  • the_title()
  • the_content()
  • the_permalink()
  • get_post_type_archive_link('post_type') : renvoie l'url de l'archive du post_type passé en argument
